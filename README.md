# Hotspot-Generator
Using TP_Link Atheros USB Dongle, this will set up a chosen SSID name, generate random password and allow one to connect to wifi via QR code.

# How to

1. Set up the hotspot.
<img src="https://raw.githubusercontent.com/HLOverflow/Hotspot-Generator/images/img/1_setup.png" width="40%">

2. Connect to wifi hotspot with Android.
<img src="https://raw.githubusercontent.com/HLOverflow/Hotspot-Generator/images/img/2_android_scanqr.jpg" width="40%">

3. Stop with Ctrl-C.
<img src="https://raw.githubusercontent.com/HLOverflow/Hotspot-Generator/images/img/3_Stopping.png" width="40%">
